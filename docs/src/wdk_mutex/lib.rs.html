1//! An idiomatic Rust mutex type for Windows kernel driver development.
2//!
3//! The crate will safely check IRQL before doing operations which would cause a STOP CODE of
4//! IRQL NOT LESS OR EQUAL (**except for RAII dropping of scoped Mutex Guards**).
5//! In those cases, the API will return an error of the internal type `DriverMutexError`.
6//!
7//! This crate is a work in progress to implement additional mutex types and functionality as required. Contributions, issues,
8//! and discussions are welcome.
9//!
10//! This crate is **not** affiliated with the WDK crates provided by Microsoft, but is designed to work with them for Windows Rust Kernel Driver
11//! development.
12//!
13//! # Tests
14//!
15//! Tests have been conducted on public modules.
16//!
17//! All tests are carried out at [wdk_mutex_tests](https://github.com/0xflux/wdk_mutex_tests),
18//! a separate repository which is built as a driver to test all functionality of the `wdk-mutex` crate. If you wish to run the tests
19//! yourself, or contribute, please check that repository.
20//!
21//! <section class="warning">
22//! As this crate integrates into the wdk ecosystem, Microsoft stipulate: This project is still in early stages of development and
23//! is not yet recommended for production use.
24//!

26//! </section>
